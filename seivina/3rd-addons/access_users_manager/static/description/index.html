A complete solution for user rights management

Access User Manager is an Odoo application designed to streamline and simplify user access management. It provides an intuitive interface allowing administrators to control various access rights without requiring complex technical knowledge.

With this tool, you can:

  • Hide specific menus, buttons and tabs
  • Manage model access rights (create, edit, delete, view, actions, archive/unarchive, duplicate and export)
  • Define field access rights (make fields invisible, required or read-only)
  • Configure domain-based access rights, allowing conditional restrictions
  • Make any user read-only, hide discussion thread or disable developer mode
