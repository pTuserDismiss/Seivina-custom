===================================
Access User Access Manager
===================================

Simplify and centralize user access management in Odoo with the **Access User Access Manager** module. This tool empowers you to customize and manage user permissions, menu visibility, field access, button actions, and domain-specific rules â€” all without requiring intricate technical knowledge.

Key Features
------------

- **Menu Management**
  - Hide or restrict access to menus, submenus, and buttons.
  - Multi-company support for menu customization.

- **Field Access Control**
  - Hide, make read-only, or set fields as required.
  - Control access to specific fields based on roles or groups.

- **Role-Based Permissions**
  - Granular access control for different user roles and groups.
  - Configure security roles with ease.

- **Domain-Specific Restrictions**
  - Set conditional access rules based on field values.
  - Restrict or allow actions like create, edit, delete, or view.

- **Advanced View and Action Control**
  - Hide or restrict views (tree, form, kanban, pivot, graph, etc.).
  - Manage access to reports, chatter, and smart buttons.

- **Centralized Dashboard**
  - Unified control panel for managing user access settings.
  - Track and adjust permissions from a single interface.

Benefits
--------

- Enhance data security by restricting unauthorized access.
- Boost productivity with user-specific and role-based interfaces.
- Simplify the configuration of access rights without technical expertise.

How It Works
------------

1. **Setup Roles and Permissions**
   - Define roles and assign permissions to users or groups.

2. **Customize Access Rules**
   - Configure menus, fields, and actions to match organizational needs.

3. **Centralized Monitoring**
   - Use the dashboard to review and update permissions as needed.

Installation
------------

1. Install the module via the Odoo App Store or by uploading the module files.
2. Ensure dependencies like `openpyxl`, `werkzeug`, and `lxml` are installed.
3. Activate the module in the Odoo backend.

Compatibility
-------------

- Odoo 18.0 and above
- Do not supports multi-company setups.
