How does it work

Consulting Contact us to learn more about the module.*

Purchase You purchase the module after the consultation.

3

Installation Install the module on your server following our instruction.**

4

Configuration Custom the customer number individually in General Setting and trigger the generating manually.***

*The module is compatible with standard Swiss localisation. If you have already made extensive customisations, please let us know during the consultation so that we can find the best solution for your requirements.

**Alternatively, we can install the module for you at additional cost.

***After installation, the module will not generate customer numbers automatically. If you only want to generate numbers for new customers moving forward, simply check the box “Generate number for new customers” and save. To assign numbers to existing customers as well, click the “Generate Number for Existing Customers” button and save. If you wish to customize the number format, fill in the relevant fields. When prompted, click Save first, then press the button to apply your changes. If no custom data is entered, the module will use the default format to generate customer numbers.
