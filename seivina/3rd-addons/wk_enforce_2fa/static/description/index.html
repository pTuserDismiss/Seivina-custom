How Does Odoo Ensure 2FA?

The Odoo Enforce 2FA module enables administrators to enforce Two-factor authentication (2FA) for public users, improving account security and preventing unauthorized access. A popup will appear on various pages like the cart page, login page, checkout page, etc requiring users to enter their login and password, and a 2FA PIN. Users must reload the page to revert to normal login functionality.

Why Enforce 2FA?

Two-factor authentication (2FA) is a crucial security measure that verifies a user's login request by confirming it with another user. It is commonly used in multi-factor transactions like phone number codes. 2FA prevents compromised passwords and encourages digital safety awareness. It encourages collaboration between users and administrators, emphasizing the importance of security in every login or transaction, unlike other passive security methods.

Enforce 2FA For Portal User

  • The portal user is unable to log in without a 2FA.
  • The addition of products to the cart is not possible without the use of 2FA.
  • Users cannot Proceed with the Checkout Process Without 2FA.
  • Odoo Enforce 2FA forces portal users to complete 2FA.

2FA QR Code Pop-Up

  • After adding the username and password 2FA PIN will be asked to enter.
  • After the correct PIN, the user will log in normally.
  • You will get a unique code whenever a user tries logging in.
  • Unique code will ensure data security and confidentiality.
